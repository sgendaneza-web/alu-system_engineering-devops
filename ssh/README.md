# SSH
This project covers secure remote server access using SSH and RSA keys.

## Files
| File | Description |
| --- | --- |
| `0-use_a_private_key` | Connects to the assigned server using `~/.ssh/school`. |
| `1-create_ssh_key_pair` | Creates a 4096 bit RSA key pair named `school`. |
| `2-ssh_config` | Uses the school private key and refuses password authentication. |
